# PropertyOps (MVP)

This repo contains the Django models for an MVP property management backend.

## Run locally (recommended via Gitpod or Replit)

- Create a virtualenv, install from `requirements.txt`
- `python manage.py migrate`
- `python manage.py createsuperuser`
- `python manage.py runserver`
Note: For iPad, use Gitpod (gitpod.io/#REPO_URL) or import into Replit.
